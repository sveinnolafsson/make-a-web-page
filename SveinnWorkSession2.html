Inheritance

Properties are applied to an element and all its descendants.
Box-related properties are not inherited.
Main properties inherited are color, font, spacing, line height, list styles, text-align, text indenting, white space and word-spacing.
Properties that are not inherited are background, border, height and width plus min- and max height and width, margin outline, padding, vertical-align and more.

Box sizing in css

An easy tool is available in css:

*{
box-sizing: border-box;
}

Older browsers may not handle this, so add
browser-specific prefixes, like this:

*{
-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
-ms-box-sizing: border-box;
box-sizing: border-box;
}

You can set size with percentage, in which case it resizes according to the screen or box size.
You can set size with pixels, in which case it is set.
You can also have max and min value


Main issues learned from stage 1:

  • Use structured code, so that the computer understands it
  • Maintain clarity in code, so it is easier to check the code, fix error or change the code
  • Be consistent, so that same style and same class is coded the same way
  • Avoid repetition. Use one set of instructions for the same thing throughout, so for instance to declare the same style of format in your file, declare it once and refer to it, do not repeat it.
